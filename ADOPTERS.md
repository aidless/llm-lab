## How to add yourself

Edit this file, add a row in one of the tables, and open a PR.
For listed organisations, keep the description to one short
line; we'll link out for details in a future case-study file
once we have at least 3 listed users (per the
`GOVERNANCE.md` "core team" trigger).

For pilot users, the "What they're testing" column can be as
specific as you like — the more concrete, the more useful the
list is to other pilots.
